# Desafio_Contador
### Descrição do Projeto

O projeto **DesafioControleFluxo** implementa um sistema de contagem baseado em parâmetros fornecidos pelo usuário. Desenvolvido em Java, o programa recebe dois números inteiros via terminal e realiza a impressão de uma sequência de números incrementados, demonstrando o uso de estruturas de controle de fluxo. A lógica de contagem é executada apenas se o primeiro número for menor que o segundo; caso contrário, uma exceção personalizada é lançada para garantir a integridade dos dados de entrada.

A classe `Contador` é responsável pela leitura dos parâmetros e pela execução da contagem, enquanto a classe `ParametrosInvalidosException` representa a exceção de negócio, garantindo que o segundo parâmetro seja sempre maior que o primeiro. Este projeto serve como exercício prático para reforçar o entendimento de controle de fluxo, manipulação de exceções e interação com o usuário em Java.
